# SparkTest

Code Deliverable for a cloud-connected, temperature sensing fridge monitor.
## Functional Elements
- Interrupt Based Scheduler
- WiFi Connectivity
- Active Buzzer
- Push-Button door sensor
- HT11 Temperature & Humidity Sensor
